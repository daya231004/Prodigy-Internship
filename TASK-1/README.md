# 🔐 Secure User Authentication System

## 📌 Task - 01 | Prodigy Infotech Internship

This project implements a secure user authentication system with login and registration functionality.  
Users can create an account, log in securely, and access protected routes only after successful authentication.

---

## 🚀 Project Overview

The main objective of this project is to build a secure authentication system using modern backend technologies.  

The system includes:

- User Registration (Sign Up)
- User Login (Sign In)
- Password hashing using bcrypt
- JWT-based authentication
- MongoDB database integration
- Protected API routes
- Frontend login & signup interface (DAYZ Clothing theme)

---

## 🛠️ Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose**
- **bcrypt.js**
- **jsonwebtoken (JWT)**
- **HTML & CSS**

---

## 🔒 Security Features

- Passwords are securely hashed using bcrypt before storing in the database.
- JWT tokens are generated after successful login.
- Protected routes can only be accessed using valid authentication tokens.
- Environment variables are used for secret keys.
